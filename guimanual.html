GUI Manual

To run swift GUI interface, use swift.py script. 
cd path/to/swift/
./swift.py

MAIN WINDOW LAYOUT

UI Elements

1 - Source file text box
Write or select (the left Select button) the path to the source file.
2 - Target file text box
Write or select (the right Select button) the path to the target file (can't be the same as source file).
3 - Source data table
Display the source data as a table, use button Preview.
4 - Target data table
Display the target data as a table if the Preview after convert checkbox is checked.
5 - Infobar.
Show an info messages.
6 - Source Select button
Select the path to the source file.
7 - Target Select button
Select the path to the target file.
8 - Convert button
Start a conversion.
9 - Source Settings button
Open the settings dialog for the source file.
10 - Source Original data button
Open the Original data dialog with the text source data.
11 - Preview button
Display a preview of the source data in the Source data table.
12 - Info button
Export statistics of the source data into a selected file.
13 - Target Settings button
Open the settings dialog for the target file.
14 - Target Original data button
Open the Original data dialog with the text source data.
15 - Preview after convert checkbox
If is chacked and if conversion was successful, the target data will be displayed in the target data table.
16 - Split bar
Allows to change width of source/target data table.

SHORTCUTS

  • Ctrl+s, Ctrl+c: Start convert.
  • Ctrl+c: Copy selected data in a table to the clipboard.
  • Ctrl+p: Preview.
  • Ctrl+i: Info.
  • Ctrl+f: Find in the source/target data table.
  • Ctrl+s, Ctrl+f: Select the source file.
  • Ctrl+t, Ctrl+f: Select the target file.
  • Ctrl+s, Ctrl+s: Open the source settings.
  • Ctrl+t, Ctrl+s: Open the target settings.
  • Ctrl+s, Ctrl+o: Show the source original data.
  • Ctrl+t, Ctrl+o: Show the target original data.
  • Atlt+s: Set focus to the Source file text box.
  • Atlt+t: Set focus to the Target file text box.
  • Alt+arrowleft: Move the split bar to the left.
  • Alt+arrowright: Move the split bar to the right.

EXAMPLE

Conversion CSV (example.csv) to CXT (result.cxt)

Main window

  1. Select the source file
  2. Show a preview of data.
  3. Open the source settings dialog and fill relevant arguments.
  4. Select the target file
  5. Open the target settings dialog and fill relevant arguments.

The source settings dialog

  1. Set the Target Attributes argument.
  2. Set the Missing Value argument.
  3. Confirm the form by click on the button OK.

The target settings dialog

  1. Set the Objects argemnts.
  2. Confirm the form by click on the button OK.

Finish

  1. Click on the Convert button.
