Welcome

on a home page of project Swift - Relational Data Converter.

Swift is an open source, cross platform project written in Python, that converts between the file formats used for example in Formal Concept Analysis (FCA) (DAT, CXT, DTL) and public repositories such as UCI Machine Learning Repository (ARFF, CSV, C4.5).
